# Snake

Pequeño juego de Snake escrito en HTML/CSS/JS.

## Ejecutar

Abrir `index.html` en un navegador (doble clic), o servir con un servidor HTTP simple:

```bash
python -m http.server 8000
```

y abrir `http://localhost:8000`.

## Estructura

- `index.html` - entrada del juego
- `snake-game.js`, `snake-logic.js` - lógica del juego
- `snake.css` - estilos
